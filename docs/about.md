# 关于平台

## 平台介绍

文苑推免指南是一个专门为中文系保研学生打造的信息聚合平台。我们致力于收集、整理全国985/211高校的保研推免信息，帮助广大保研学子更高效地获取最新、最准确的招生资讯。

## 我们的使命

- **信息透明**：打破信息差，让所有保研学生都能平等获取优质院校信息
- **数据准确**：所有信息均来自官方渠道，确保真实可靠
- **更新及时**：第一时间更新各校推免通知，不错过任何重要时间节点

## 收录范围

目前平台收录了：

- **985高校**：39所
- **211高校**：24所
- **覆盖专业**：文艺学、中国古代文学、中国现当代文学、汉语言文字学、比较文学等主流中文专业

## 使用说明

### 浏览院校
1. 访问「院校名录」页面
2. 使用顶部筛选按钮按类型（985/211）筛选
3. 点击任意院校查看详细信息

### 搜索功能
- 支持按学校名称搜索
- 支持按专业方向搜索
- 实时显示搜索结果

### 收藏功能
- 点击卡片上的心形图标收藏院校
- 收藏信息保存在本地，下次访问自动加载

## 数据来源

所有院校信息均来自以下官方渠道：

- 各高校文学院官网
- 各高校研究生院招生网
- 各高校官方招生通知
